URL

  • URLs descriptifs : Utilisez une URL de page descriptive, qui doit refléter le mot clé que vous avez ciblé.
  • Extension de fichier - Ne supprimez pas l'extension de fichier dans les URL.
  • HTTPS - La sécurité est une priorité absolue pour Google.
  • Traits d'union - Divisez les mots à l'aide de traits d'union.
  • Localisation - Choisissez un domaine spécifique au pays, pour de meilleurs résultats de recherche locale.
  • Sous-domaine ou sous-dossier - Les sous-domaines sont considérés comme des domaines distincts.
  • Créateur d'URL - Utilisez cet outil pour ajouter des paramètres de campagne personnalisés à vos URL.

Accessibilité

  • 403 : Fournir une page 403 - Accès refusé.
  • 404 : Fournir une page 404 - Page non trouvée.
  • Recherche personnalisée - Grâce au champ de recherche Google Sitelink, les internautes peuvent atteindre votre contenu plus rapidement.
  • Fichier non trouvé : Évitez les erreurs 404 FILE_NOT_FOUND.
  • Mise en page : Utilisez des divs plutôt que des tableaux pour la mise en page. L'utilisation de tableaux n'est pas sémantiquement correcte.
  • Déplacer un site web : Redirigez tous vos liens vers le nouvel emplacement via .htaccess.
  • Pagination - Implémentez les attributs rel="next" et rel="prev" aux liens.
  • Performances - Les performances et le temps de chargement sont importants.
  • Redirections : Évitez les redirections si possible. Utilisez la redirection 301 plutôt que 302.
  • RichSnippets - Marquez votre code avec des snippets riches, ils s'affichent sur la page des résultats de recherche.
  • Robots - Bloquer les pages qui ne doivent pas être indexées via le fichier robots.txt ou <meta name="robots" content="">.
  • Validation : Écrivez du code valide(validateur HTML, validateur CSS).
  • WAI-Aria - Utilisez les balises WAI-Aria pour aider les machines à comprendre votre code.

Informations Méta

  • Description - Chaque page doit avoir une description unique (160 caractères maximum). <meta name="description" content="">.
  • Titre : Chaque page doit avoir un titre parlant unique (60 - 100 caractères) <titre>Titre du site web</titre>.

Mots clés

  • Contenu : Le mot-clé doit apparaître dans ~3% de la longueur de l'article.
  • En-tête : Le mot-clé doit apparaître dans les titres.
  • Métabalise - Vous pouvez omettre la balise <meta name="keywords" content="">, les moteurs de recherche n'utilisent pas cette balise méta.
  • Recherche : Classez les mots clés ayant un trafic élevé et une concurrence moindre.
  • Unique : Chaque page doit comporter un seul et unique mot clé ciblé.
  • Titre : Le mot-clé doit apparaître dans le titre de la page.
  • URL - Le mot-clé doit apparaître dans le nom de l'URL.

Contenu

  • Le contenu : Le contenu est ce qui compte le plus en matière de référencement.
  • Flash : évitez le contenu et les pages en Flash. Ils ne sont pas accessibles sur les téléphones mobiles et seront moins bien classés.
  • La fraîcheur : Le nouveau contenu est important. Il est recommandé de mettre à jour les pages ou de poster régulièrement.
  • Titres : Structure claire H1 -H6 max. 70 caractères de long.
  • Longueur : L'article doit comporter au moins 300 mots.
  • Fort : Utilisez une balise forte pour mettre en évidence votre mot-clé ciblé.
  • Unicité - Ne fournissez pas de contenu dupliqué, utilisez des types de contenu uniques.

Images

  • Balise Alt - Ajoutez une balise Alt, c'est-à-dire une description de l'image (60 à 70 caractères).
  • Dimensions : Ajoutez les attributs width="" et height="" à l'image.
  • Nom du fichier - Utilisez un nom court et descriptif.
  • Optimisation - Optimisez les images en supprimant certaines informations méta.
  • Images réactives - Servez l'image la plus optimisée correspondant à la taille de la fenêtre.
  • Taille : gardez la taille des fichiers aussi faible que possible.

Vidéos

  • Contrôles : Ajoutez des commandes pour lire et contrôler votre vidéo.
  • Intégrer : Autorisez les autres à intégrer vos vidéos.
  • Transcriptions : Utilisez les transcriptions pour l'indexation, la facilité d'utilisation et le contenu.
  • Contenu injouable - Évitez les contenus vidéo injouables. Utilisez la balise HTML5 <video> au lieu de Flash.

Liens

  • Backlinks : N'ajoutez des liens externes que si vous avez obtenu un lien retour vers votre site.
  • Liens internes : Ajoutez ~3 liens internes à votre contenu.
  • Langues - La balise hreflang indique à Google la langue utilisée sur une page spécifique. page spécifique, afin que le moteur de recherche puisse proposer ce résultat aux utilisateurs qui effectuent des recherches dans cette langue <link rel="alternate" href="example.com/fr/" hreflang="fr-fr" />.
  • Nommage : Utilisez un nom de lien descriptif : "Cliquez ici" ou "En savoir plus" sont de mauvais textes de lien. Mieux vaut "En savoir plus sur le référencement et l'accessibilité du Web".
  • nofollow - Ajoutez l'attribut rel="nofollow" aux liens externes uniquement pour éviter le spam et les mauvais liens.
  • Title : ajoute l'attribut title aux liens.

Mobile

  • AppLinks - Les applications qui renvoient à votre contenu peuvent ensuite utiliser ces métadonnées pour établir un lien profond avec votre application.
  • convivial pour les mobiles - Les sites optimisés pour les mobiles sont marqués dans les résultats de recherche. Testez un site adapté aux mobiles.
  • Smart App Banner - Safari dispose d'une fonction Smart App Banner qui offre une méthode standardisée de promotion des applications sur l'App Store à partir d'un site web.
  • Cibles cliquables - Les liens cliquables ne doivent pas être trop petits.
  • Viewport : Indiquer aux navigateurs comment ajuster les dimensions et la mise à l'échelle de la page en fonction du périphérique. <meta name="viewport" content="width=device-width, initial-scale=1">

sitemap

  • Sitemap HTML - Un sitemap HTML permet aux visiteurs de naviguer facilement sur un site web.
  • Sitemap des images - Augmente la visibilité de vos images dans les résultats de la recherche d'images.
  • Plan de site mobile - Pour les téléphones fonctionnels, vous pouvez créer un plan de site mobile.
  • Sitemap vidéo - Assurez-vous que les moteurs de recherche connaissent l'ensemble du contenu vidéo de votre site.
  • Plan de site XML - Aide les moteurs de recherche à indexer vos pages.
